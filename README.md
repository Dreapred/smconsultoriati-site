# S&M Consultoria em TI — Site Institucional

Site estático com as páginas institucionais da **S&M Consultoria em TI**, incluindo apresentação de serviços, contatos e as páginas de Termos de Uso e Privacidade. O código está pronto para ser publicado em qualquer hospedagem estática ou CDN.

## Estrutura
- `index.html` — Página principal com seções institucionais e contato.
- `termos.html` — Termos de Uso.
- `privacidade.html` — Política de Privacidade.
- `style.css` — Estilos globais.
- `script.js` — Comportamentos básicos de interface.
- `img/` — Logos e ícones utilizados no site.

## Como visualizar
Abra `index.html` diretamente no navegador ou sirva a pasta com qualquer servidor HTTP estático (ex.: `python -m http.server 8080`).
